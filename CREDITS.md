## GNOME

The modern-looking efficient desktop environment used.

## Zed

The text editor used

## yt-dlp developers

A tool used for the music player, [Mixtapes](https://github.com/m-obeid/Mixtapes).

## [Liquid-glass Gnome Extension](https://github.com/ryohsuke1231/liquid-glass) by Ryosuke Watanabe

Replicates the Liquid Glass effect introduced in Apple's 2025 software.

## [dash-to-dock](https://github.com/micheleg/dash-to-dock)

A dock for the Gnome Shell

## [wintile-beyond](https://github.com/GrylledCheez/wintile-beyond)

Better tiling for GNOME.

## [Sunshine](https://app.lizardbyte.dev/Sunshine/)

A streaming server. Works well with a Moonlight client.

A more modern shell for UNIX

## [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)

Allows browser-like autosuggestion of commands based on past history.

## [ProtonPlus](https://protonplus.vysp3r.com/)

A modern GUI front end for managing versions of Proton.

### [Proton-GE](https://github.com/gloriouseggroll/proton-ge-custom)

An up-to-date browser without the various ideas Mozilla has had for Firefox.
